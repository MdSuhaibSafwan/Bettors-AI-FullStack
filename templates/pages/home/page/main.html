What is this?

Django Template was used as a base to create a chat application using OpenAI API for learning purposes.

The main functionality of this chat is in the 'apps/chat' section.

{# --- Order Revers and colsize change md or less ---#}

Streaming Output

  • Django Channels (Web Socket) is used to support streaming output.

API Parameters Configuration

  • You can change hyperparameters and icons for each room.

Others

  • You can specify the number of conversation history entries to consider in your chat.
  • You can check the number of tokens used in the conversation, both for input and output.

Installation

Show differences only with Django Template.

Configuration in config > settings.py

You can change the following default values to True/False:

  • Use RADIS
    • Not working well with GCP, under investigation/Your comments are appreciated if you have more insights. IS_USE_RADIS = False

Prepare .env File

A sample 'sample.env' file is included.

  • Set OpenAI API KEY
    • OPENAI_API_KEY='*** YOUR OPENAI_API_KEY ***'
  • If you are using Radis
    • Not working well with GCP, under investigation/Your comments are appreciated if you have more insights. RADIS_HOST='*** RADIS HOST ***'
    RADIS_PORT='*** RADIS PORT ***'

Execution

  • Install required packages
    • $ pip install -r requirements-base.txt
  • Run migration (bat file available)
    • $ ProjectSetupBat
  • Run the server
    • $ python manage.py runserver

For details, please see 'requirements-base.txt'.
